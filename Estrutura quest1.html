Definição das estruturas:

HTML é usado para criar a estrutura e o conteúdo básico da página.
CSS é aplicado para definir a aparência e o layout dos elementos HTML.
JavaScript é usado para adicionar funcionalidades dinâmicas e interativas.


* Tag h1 a h6:

- São utilizadas para definição de títulos. Sendo h1 definida para título e os subsequentes como subtítulos.

Exemplos:

Nome - exemplo do h1

Nome - exemplo do h2

Nome - exemplo do h3

Nome - exemplo do h4

Nome - exemplo do h5
Nome - exemplo do h6

* Tag p:

- É indicada para escrever pragrafo.


* Listas:

- São usadas para agrupar uma série de itens. Existem as listas não ordenadas e as ordenadas.

Exemplos:

Lista Não Ordenda

  • arroz
  • feijão
  • macarrão
  • açúcar

Lista Ordenada

  1. arroz
  2. feijão
  3. macarrão
  4. açúcar

* Tabelas:

- As tabelas são definidas com a tag table e sua estrutura é formada pelas as tags th e td. A tag th é definida como o cabeçalho e a tag td contém os dados da tabela.

Exemplo:

Aluno(a) Curso Turno
Maria Engenharia Noite
Ana Medicina Manhã
José Ads Tarde

* Formulários:

- Os formulários são criados com a tag form. Existindo o elemento label que associa um texto descritivo a um campo de entrada, imput permite a entrada de texto de uma linha.

Exemplo:

DADOS PESSOAIS


* Com que frequencia você come frutas?

* Quais desses alimentos não pode faltar na sua alimentção?

Pão Macarrão Verduras Derivados do leite Carne Refrigerante Suco natural Tubérculos Feijão Embutidos Biscoito Bolo
